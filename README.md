# NexusPay

> Secure transaction processing and comprehensive merchant dashboard.

## Overview
NexusPay is a client-side fintech dashboard designed to simulate a robust payment processing platform. It empowers merchants with a comprehensive suite of tools to track transactions, manage multi-currency payments, and visualize financial workflows in real-time.

By providing a clean, responsive interface, NexusPay solves the complexity of financial data management for small to medium-sized businesses. Target users include merchants, financial administrators, and business owners who need a streamlined view of their revenue streams.

## Tech Stack
* React
* Vite
* Tailwind CSS
* shadcn/ui
* Recharts (for data visualization)

## Features
* **Interactive Merchant Dashboard**: View revenue analytics and transaction summaries at a glance.
* **Real-time Transaction Tracking**: Filter and search through transaction history effortlessly.
* **Multi-currency Support**: Switch between currencies with simulated real-time conversion.
* **Payment Processing Simulation**: Mock checkout flow to simulate incoming transactions.
* **Persistent Local Storage**: Safely stores transaction history in the browser without needing a backend.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   ```
3. **Build for production:**
   ```bash
   npm run build
   ```
